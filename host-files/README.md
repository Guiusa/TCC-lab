# Host Lab Files

Scripts and helpers for the host container.

## Available scripts

| Script | Purpose |
|---|---|
| `ping_test.sh` | Simple ICMP ping to suricata |
| `syn_flood.sh` | hping3 SYN flood toward suricata |
| `udp_flood.sh` | hping3 UDP flood toward suricata |

Run any script with: `bash /root/host-files/<script>`
